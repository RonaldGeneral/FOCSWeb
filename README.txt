1. Download XAMPP Control Panel and Start Apache and MySQL server with it.
2. Open MySQL admin page
3. Create a new database called "sem"
4. If you have own MySQL username and password, go to db.js and enter in line 6 and 7
5. The .env file have api key of gemini api, you can use ur own gemini api key if the key does not work.
6. Go to http://localhost:3000, at the navigation bar on top, find "Tools" to link to the 3 modules that we developed. 

Elective Course - http://localhost:3000/elective
Requirement Checker - http://localhost:3000/req
Compare Programme - http://localhost:3000/compare
